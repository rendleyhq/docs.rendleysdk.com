    Class SubtitlesClip

    Sprite-less clip wrapper that binds a library subtitles asset into the timeline.

    Rendering and hit-testing are delegated to SubtitlesManager, so the clip itself mainly carries timing, subtitles linkage, and editor interaction hooks.
