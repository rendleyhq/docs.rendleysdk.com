# Clips

The clip represents the smallest unit of the composition. It can either be a text, an image, a video, an audio or even a custom HTML element.

## Supported Clips

We currently support the following clips:

### Image

Responsible for displaying images

| Format | Support |
| ------ | :-----: |
| JPEG   |   ✅    |
| PNG    |   ✅    |
| GIF    |   ✅    |
| WEBP   |   ✅    |
| BMP    |   ✅    |
| HEIC   |   ✅    |

### Video

Responsible for displaying videos

| Format | Support |
| ------ | :-----: |
| MP4    |   ✅    |
| MOV    |   ✅    |
| AVI    |   ✅    |
| WEBM   |   ✅    |
| MKV    |   ✅    |
| M3U8   |   ✅    |

### Audio

Responsible for handling audio

| Format | Support |
| ------ | :-----: |
| MP3    |   ✅    |
| M4A    |   ✅    |
| OGG    |   ✅    |
| WAV    |   ✅    |
| ACC    |   ✅    |

### Shape

Responsible for rendering various shapes such as rectangles, circles and more

### Gif

Responsible for rendering gifs. It can also be used for rendering stickers as it supports transparency

### Text

Create and display text elements

### HTML Text

If you have a custom HTML element, you can use this clip

### Lottie

If you want to use Adobe After Effects compositions, it can be done through this clip

### Custom

You can implement your own clips for stuff we don't support, such as Waveforms
